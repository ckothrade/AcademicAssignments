*Assignment: Assignment 2, EmployeeWorkHours

*Description: To allow the user to designate a number of employees, the number of days they worked in the week, 
the hours they worked each day and then display the employees in descending order for their total number of hours worked in the week.

*USER INSTRUCTIONS: Designate the number of employees you would like, but an employee cannot work more than 7 days out of the week, and an outofbounds error will occur if more than 7 days are entered. Also, some employees may be displayed twice if there are more than one employee who works the same amount of hours.
